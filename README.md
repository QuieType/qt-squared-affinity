Squared Affinity is going to be the name of a small super-hard Crosscode dungeon I'm modding. Right now, it just has a single room. This is the first release that I'm asking others to test, and it is very much a work in progress.
You can get to the puzzle from a teleporter on the bottom floor of the Info Hub in rookie harbor.

WARNING:
This mod currently breaks all ice physics on ice tiles (not entities) in the game. Disable the mod when you're not using it directly.
